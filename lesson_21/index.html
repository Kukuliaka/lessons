Задача №2

  • const userName = 20
  • userName = "Іштван"
Відповідь: буде помилка, тому що не можна змінювати значення константи

Задача №3

Що потрапить в консоль?

  • let numOne = 20
  • let numTwo = "20"
  • let numSumm = numOne + numTwo
  • console.log(numSumm)
Відповідь: 2020, тому що оператор додавання не змінює тип даних, а число + рядок = 2020
